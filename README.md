# Active-Directory-Actions

# Objectives
The project was to showcase different actions or events within active directory that would typically take place on a day to day basis. This project is to show mastery of these tasks.

# Tools

1) Virtualbox
2) Windows 2019 Server

# Skills Learned 

1) User creation
2) Password Creation
3) Disable User
4) Password Reset
5) Employee Logon Access

In walkthrough I created a user by the name of Michael Kyle and went through the prompts to create new user including the temporary password. Once the user logon they will be prompted to change the password.
